# Netflix EDA — Exploratory Data Analysis

## Overview
Exploratory Data Analysis on Netflix Movies & TV Shows dataset using Python and Pandas.

## Dataset
- Source: Kaggle — Netflix Movies and TV Shows
- 8807 titles with 12 columns

## Tools Used
- Python
- Pandas
- NumPy

## What I Analyzed
- Movies vs TV Shows distribution
- Top countries producing Netflix content
- Content growth over the years
- Most popular genres
- Top directors and cast members
- Longest movies on Netflix

## Key Findings
- United States dominates Netflix with the most content
- 2019 was the peak year for content added on Netflix
- International Movies is the most common genre
- Anupam Kher is the most featured actor on Netflix
- The longest movie on Netflix is The School of Mischief at 253 mins
